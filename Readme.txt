# Logistics API Integration with Express.js

Menambah validasi token ninja ada di database, tidak lagi menyimpan di local storage
Menambah score ekspedisi

This application integrates with four logistics APIs: Logistic Ninja, JNE, SAP, and ID Express, using Express.js.

## Usage

This application allows users to interact with the logistics APIs via the following endpoints:

- Logistic Ninja: `/logistic-ninja`
- JNE: `/jne`
- SAP: `/sap`
- ID Express: `/id-express`

## Installation

1. Clone the repository.
2. Navigate into the project directory.
3. Install dependencies.
4. Create a `.env` file with necessary API keys.
5. Start the server.
